QWEP API — это своеобразный посредник между сайтами поставщиков и вашими сервисами. Сделав запрос к API, инициируется процесс поиска по заданным параметрам по вашим аккаунтам на сайтах поставщиков. После завершения процесса сбора данных, вы получаете структурированный ответ.

Скорость поиска ограничивается скоростью самого медленного сайта поставщика.

Система поддерживает асинхронный режим, поэтому ваши пользователи могут не ждать полного завершения процесса поиска, а получать данные уже в первые секунды после старта.

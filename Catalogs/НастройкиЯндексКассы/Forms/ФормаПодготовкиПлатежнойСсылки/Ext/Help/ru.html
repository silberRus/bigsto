Форма оплаты через Яндекс.Кассу

Позволяет подготовить ссылку для оплаты счета через Яндекс.Кассу и отправить ее покупателю.

Для получения платежной ссылки необходимо нажать кнопку Сформировать ссылку. Данные документа будут отправлены в сервис выставления счетов. Сервис вернет ссылку, которую необходимо выслать покупателю. Перейдя по ней на страницу оплаты, клиент сможет оплатить счет.

Способ доставки чека

Если в настройках Яндекс.Кассы в разделе Как пробивать чеки выбрано Через Яндекс.Кассу, для соблюдения 54-ФЗ требуется указать контактные данные для доставки электронной копии чека: электронная почта или SMS. При самостоятельном режиме работы с онлайн-кассой эти данные не заполнятся.

Контактные данные покупателя (адрес электронной почты, телефон) автоматически заполняются из карточки партнера (покупателя) (закладка Адреса, телефоны).

Ссылка для оплаты счета покупателем

Для получения платежной ссылки необходимо нажать кнопку Сформировать ссылку.

Если счет был изменен уже после формирования ссылки, необходимо обновить информацию о документе на стороне сервиса нажав на кнопку Обновить данные.

Отправка ссылки покупателю

Платежную ссылку можно отправить покупателю по электронной почте или SMS нажав на кнопку Отправить ссылку. Ссылку можно передать используя другие средства связи, скопировав ее в буфер обмена.

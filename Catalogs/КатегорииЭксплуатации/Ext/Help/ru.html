Категории эксплуатации

Классификатор вариантов эксплуатации товарно-материальных ценностей. Предназначен для объединения спецодежды, спецоснастки, инвентаря и прочих хозяйственных принадлежностей, по общим признакам учета в эксплуатации и нормам погашения стоимости

  • Флаг необходимости ведения Инвентарного учета для оборудования данной категории;
  • Способ погашения стоимости в Бухгалтерском и Налоговом учете позволяет выбрать методику, по которой будет списана стоимость передаваемых в эксплуатацию позиций;
  • Срок эксплуатации, вместе с периодом полезного использования, задает интервал времени линейного списания, при установленном способе погашения стоимости По сроку эксплуатации;
  • Для методики погашения стоимости по наработке необходимо заполнить поля: Объем наработки и Единица измерения наработки;
  • В поле Статья расходов указывается статья, используемая в документе передачи в эксплуатацию по умолчанию для данной категории;
  • Описание - многострочный комментарий, присутствующий форме выбора категории как отдельное поле.

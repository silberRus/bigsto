Настройки ЭДО

Справочник Настройки ЭДО используется для указания параметров обмена электронными документами с контрагентами.

Параметры определяются для организации, контрагента, договора и вида документа ЭДО. В справочнике устанавливается основной Профиль настроек ЭДО организации, идентификатор контрагента (строковое значение идентификации при обмене), форматы электронных документов, форматы пакетов электронных документов, а также технические настройки способа обмена (параметры авторизации, каталоги и другое). Дополнительно может быть настроено шифрование данных и проверка сертификатов контрагента для входящих документов.

Настройка ЭДО может производиться в простом и расширенном режиме. В простом режиме настройки указывается один профиль настроек ЭДО организации, в расширенном, профиль настройки ЭДО может настраиваться для каждого вида документа. Необходимость настройки расширенного режима, может возникнуть при использовании нескольких способов обмена с контрагентом по видам документов.

Сервисными возможностями являются тестирование настроек для технической проверки способа обмена и использование команд ЭДО для отправки каталога товаров контрагенту и просмотру списка ранее отправленных каталогов, а так же текущего статуса обмена.

Профили настроек ЭДО

Справочник Профили настроек ЭДО используется для настройки параметров обмена электронными документами для организации.

Обмен электронными документами может осуществляться через оператора электронного документооборота (далее оператора ЭДО) или напрямую между контрагентами.

Профиль настроек ЭДО можно создать, используя помощник двух видов

В форме элемента справочника заполняются реквизиты

Профиль настроек ЭДО можно создать, используя помощник двух видов
  • помощник подключения к сервису оператора ЭДО;
  • помощник настройки прямого обмена с контрагентами.

В форме элемента справочника заполняются реквизиты
  • Организация - выбор организации, для которой производится настройка параметров;
  • Идентификатор организации - строковое значение идентифицирующие организацию при выполнении обменов, для обмена через оператора ЭДО устанавливается автоматически;
  • Способ обмена - способ обмена в настройке (через оператора, прямой обмен);
  • Виды электронных документов - поддерживаемые в настройке параметров обмена виды и форматы документов;
  • Сертификаты организации - сертификаты электронной подписи используемые для обмена документами;
  • Дополнительные реквизиты - реквизиты для настройки соединения (имена каталогов, серверов, параметры авторизации), заполняются в соответствии со способом обмена.

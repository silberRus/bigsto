Номера ГТД

Справочник предназначен для хранения номеров всех грузовых таможенных деклараций, по которым когда-либо поступали импортные товары. Для каждого номера ГТД регистрируется информация о стране происхождения, из которой поступил импортный товар. Информация о стране происхождения может быть заполнена из классификатора Страны мира.

Возможность оформления операций закупки товаров по импорту с оформлением таможенной декларации определяется функциональной опцией Импортные закупки в разделе НСИ и администрирование - Закупки - Импорт.

Предусмотрено два варианта поставки импортных товаров:

  • Если в системе регистрируется прохождение товаров через таможню (оформляется таможенная декларация), то номер ГТД и страна происхождения регистрируется непосредственно в таможенной декларации. При этом может быть зарегистрирован любой товар, независимо от установленного признака Вести учет ГТД. Поступление товаров в этом случае оформляется документом Приобретение товаров и услуг или Поступление товаров с установленной операцией закупки по импорту.
  • Если регистрируется поставка импортного товара, который уже прошел таможенное оформление, то информация о ГТД и стране происхождения указывается в документе Приобретение товаров и услуг или Поступление товаров. В документе может быть установлен любой вид операции, кроме Закупки по импорту. При этом номера ГТД и страна происхождения указываются в табличной части документа поставки только для тех товаров, для которых установлен флаг Вести учет ГТД.


В момент отгрузки товаров (оформления документа Реализация товаров и услуг) информация о списанных номерах ГТД заполняется автоматически: списывается остаток по тому номеру ГТД, который имеет более раннюю дату поставки.

Информация о номерах ГТД и странах происхождения используется при оформлении счетов-фактур.

Система позволяет контролировать заполнение номеров ГТД для импортных товаров. Контроль производится для тех товаров, для которых установлен флаг Вести учет ГТД.

Контроль заполнения номеров ГТД осуществляется в разделе НСИ и администрирование - Закупки - Импорт:

  • Запретить операции с товарами без номера ГТД - блокирует отгрузку (списание) импортных товаров, которым не присвоен номер ГТД.
  • Запретить поступление товара без номера ГТД - контроль документов Приобретение товаров и услуг, в которых отражается поступление импортных товаров. Контроль не производится в документах с установленным видом операции Закупка по импорту. Информация о ГТД для таких закупок фиксируется в момент оформления таможенной декларации.
  • Контролировать остатки номеров ГТД при импорте по схеме "товары в пути" - контроль документов Поступление товаров с операцией "товары в пути", в которых отражается поступление импортных товаров.

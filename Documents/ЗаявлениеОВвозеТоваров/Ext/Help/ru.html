Заявление о ввозе товаров из ЕАЭС

Рабочее место предназначено для формирования документов Заявление на ввоз товаров из ЕАЭС.

Документ предназначен для отражения в учете операции ввоза товаров из государств-членов Евразийского экономического союза и формирования необходимой отчетности для ФНС:

  • Заявление о ввозе товаров и уплате косвенных налогов
  • Статистическая форма учета перемещения товаров во взаимной торговле Российской Федерации с государствами-членами Таможенного союза


Работа с документом возможна при включенной опции Импортные закупки в разделе НСИ и администрирование - Закупки - Импорт.

Документ может быть составлен на неограниченное количество поставок от одного контрагента за отчетный период (месяц).

Создание документа


Для создания заявлений на ввоз необходимо:

  • В заголовке рабочего места выбрать организацию
  • На закладке К оформлению отобразится список поступлений товаров по которым необходимо создать заявления о ввозе из ЕАЭС. В этот список попадают поступления с видом хозяйственной операции Ввоз из ЕАЭС, по которым еще не создано заявлений.
  • После выполнения команды Сформировать заявление о ввозе товаров будут сформированы заявления о ввозе по выбранным накладным.
  • На закладке Заявления отображается список всех ранее сформированных документов.

Заполнение документа


Табличная часть документа заполняется по данным документов Приобретение товаров и услуг с видом хозяйственной операции Ввоз из ЕАЭС. С помощью команды Добавить открывается диалоговое окно со списком поступлений по которым не было оформлено ранее заявлений о ввозе товаров. При выборе накладной, все ее строки переносятся в заявление о ввозе.

Для формирования отчетных форм в документе дополнительно указываются:

  • код ТН ВЭД;
  • количество по ТН ВЭД, если для кода ТН ВЭД задана дополнительная единица измерения (не килограммы);
  • вес товара;
  • код вида транспорта;
  • реквизиты товаросопроводительных документов;
  • реквизиты спецификаций и участников сделки (при непрямых поставках, например при покупке товаров через комиссионера);
  • и другие сведения (команда Параметры выгрузки формы документа)


Для вывода в печатных формах информации об акцизах необходимо включить флажок Акцизы. В табличной части документа будут отображены соответствующие колонки для заполнения.

Работа с документом


Работа с документом состоит из следующих этапов:

  • Формирование и проведение документа. При проведении документа производится начисление налога на добавленную стоимость, подлежащего уплате,
  • Формирование печатных и/или электронных форм отчетов. Можно выполнить как из формы рабочего места, так и из формы документа. Если в документе указана Номенклатура поставщика то при печати и выгрузке данных наименование заполняется в терминах поставщика.
  • Оплата НДС в бюджет. Для формирования документов на оплату можно открыть рабочее место Перечисление НДС в бюджет по отдельным операциям.
  • После подтверждения оплаты НДС со стороны ФНС необходимо установить реквизиты отметки налогового органа: номер и дату подтверждении оплаты налога в бюджет, после чего работа с документом завершается. Для установки отметок нужно открыть форму по гиперссылке НДС перечислен в бюджет...

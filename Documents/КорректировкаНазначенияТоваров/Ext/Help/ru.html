Корректировка назначения товаров

Документ предназначен для корректировки назначения поступивших на склад товаров. Документ может быть оформлен только в том случае, если в разделе НСИ и администрирование - Склад и доставка - Обеспечение потребностей включена функциональная опция Обособленное обеспечение заказов.

Документ позволяет выполнить одну из следующих операций:

Резервирование

Резервирование и корректировка

Снятие резерва

Снятие резерва по многим назначениям

Заполнение документа

Резервирование

Для операции резервирования необходимо указать назначение, под которое необходимо зарезервировать товары. Для резервирования могут быть использованы товары из свободного остатка товаров на складе.

Резервирование и корректировка

Для операции резервирования и корректировки необходимо указать назначение, под которое необходимо зарезервировать товары. Для резервирования могут быть использованы товары из свободного остатка товаров на складе и товары, поступившие под другие назначения. Таким образом, данная операция позволяет переназначить обособленные остатки на складе с одного назначения на другое.

Снятие резерва

Для выполнения снятия резерва необходимо указать назначение. Документ будет заполнен обособленными остатками товаров, поступивших под указанное назначение. В результате корректировки назначения указанные товары становятся доступными для продажи на общих условиях.

Снятие резерва по многим назначениям

Для выполнения снятия резерва необходимо указать товары, поступившие на склад под назначения. В результате корректировки назначения указанные товары становятся доступными для продажи на общих условиях. Операция предназначена для массового снятия резерва товаров.
Дополнительно с помощью данной операции можно сразу зарезервировать освобождаемые при снятии товары под любые другие назначения. В этом случае товары не попадают в свободный остаток, а переназначаются из исходного назначения в новые.


Операция Произвольная корректировка назначения введена для совместимости с более ранними версиями системы.

Заполнение документа

Количество товаров, для которых нужно скорректировать назначение, склады для резервирования товаров под конкретное назначение можно уточнить в отдельном сервисе заполнения документа.

Документ может быть оформлен на основании заказа на отгрузку (заказа клиента, заказа на перемещение и т.п.) а также независимо.
В случае заполнения на основании в документе автоматически заполняется назначение, связанное с заказом на отгрузку. Операция резервирования доступна для проведенных заказов на отгрузку с установленным для товаров действием Обеспечивать обособленно или Отгрузить обособленно.
Для резервирования товаров под Направление деятельности документ оформляется независимо (из списка документов). В этом случае назначение, связанное с направлением деятельности необходимо выбрать в документе вручную.

Если в программе ведется учет в разрезе нескольких организаций, то документ сначала корректирует остатки товаров организации, указанной в документе, а затем остатки остальных организаций предприятия.
Предусмотрена также возможность фильтрации заказов по организации для последующей корректировки назначения.

При отключенной функциональной опции Разрешить обособление товаров сверх потребности (НСИ и администрирование - Настройка НСИ и разделов - Склад и доставка - Обеспечение потребностей) операции Корректировки назначения или Резервирования позволяют обособливать товары только на складах отгрузки. На операции Снятия с резерва и Снятия резерва по многим назначениям указанная опция не влияет.

Операция по Яндекс.Кассе

Документ предназначен для регистрации операций оплаты клиентом за товары, работы, услуги и возвратов оплат клиенту через сервис Яндекс.Касса.

Для оформления операций по Яндекс.Кассе необходимо предварительно создать настройку Яндекс.Кассы, указав вариант использования сервиса С договором.

Документ может быть создан 3 способами:

Вручную

Автоматически

Автоматически по расписанию

Вручную

Для ввода вручную необходимо нажать кнопку Создать, указать Контрагента, Основание платежа, Сумму операции и Сумму комиссии.

Автоматически

Для автоматического создания документа, необходимо нажать кнопку Загрузить в форме списка операций

Автоматически по расписанию

Для автоматического создания документов по расписанию, необходимо включить получение уведомлений об оплатах по расписанию в разделе НСИ и администрирование - Обмен электронными документами - Интеграция с Яндекс.Кассой и указать периодичность, с которой должны загружаться новые документы.

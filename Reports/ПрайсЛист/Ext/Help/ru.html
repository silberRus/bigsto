Прайс-лист

Отчет предназначен для формирования прайс-листа предприятия.

Особенности отчета:

  • Детализация по видам цен, используемых только при продажах данной номенклатуры;
  • Детализация только по тем номенклатурным позициям, которые есть в наличии;
  • Оперативный контроль товарных остатков организации (флаг Показывать остатки товаров).

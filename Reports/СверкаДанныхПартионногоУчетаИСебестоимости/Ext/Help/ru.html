Сверка данных партионного учета и себестоимости

Отчет предназначен для расследования проблем, возникших при выполнении этапов группы Расчет себестоимости регламентных операций по закрытию месяца.
Отчет доступен профилю Администратор при включенной функциональной опции Использовать партионный учет.

Назначение отчета
  • Позволяет ответить на вопрос: "Где возникла проблема";
  • Но, не предназначен для ответа на вопрос: "Почему возникла эта проблема".

Алгоритм формирования отчета
  • Сопоставляет данные регистров партионного учета с данными регистра Себестоимость товаров;
  • Вычисляет расхождения в данных этих регистров;
  • Выводит информацию о проблемных позициях номенклатуры с точностью до документа-регистратора.

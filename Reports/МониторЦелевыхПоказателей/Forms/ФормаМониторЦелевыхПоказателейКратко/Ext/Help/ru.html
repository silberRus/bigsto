Отображение показателей с определенным состоянием

Для отбора по состоянию доступны следующие значения:

  • Без учета состояния,
  • Цель достигнута,
  • Приемлемое состояние,
  • Неприемлемое состояние,
  • Состояние не определено,
  • Рассчитаны с ошибками.

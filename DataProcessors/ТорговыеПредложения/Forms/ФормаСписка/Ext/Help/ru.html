Торговые предложения

Рабочее место предназначено для просмотра активных соглашений с клиентами, по которым производится выгрузка торговых предложений по товарам для покупателей в сервисе 1С:Бизнес-сеть.

Для возможности использования сервиса обмена торговыми предложениями требуется включение функциональной опции Обмен данными 1С:Бизнес-сеть (НСИ и администрирование - Настройка интеграции - Обмен электронными документами).

Добавление торговых предложений в список публикуемых

Для добавления торгового предложения в список публикуемых необходимо нажать команду Добавить, откроется окно выбора Соглашений с клиентами, в котором надо выбрать нужное соглашение.

Отмена публикации

Для отмены публикации торгового предложения нужно выбрать торговое предложение в списке и нажать команду Удалить. При этом статус публикации изменится на Отмена публикации. Это действие можно отменить, нажав еще раз команду Удалить.

Синхронизация

Доступно два варианта синхронизации публикуемых торговых предложений:

  • Вручную - для этого нужно нажать команду Синхронизировать
  • Автоматически - для этого нужно установить флаг Автоматически синхронизировать и настроить расписание синхронизации. Обязательно надо настроить регламентное задание Синхронизация торговых предложений с сервисом 1С:Бизнес-сеть, указав пользователя с правами доступа к 1С:Бизнес-сеть, от имени которого будет запускаться задание.

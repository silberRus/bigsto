Сопоставление значений

В форме производится сопоставление значений текущего реквизита номенклатуры (характеристики) информационной базы со значениями сопоставленого реквизита 1С:Бизнес-сети.

Автоматическое сопоставление значений

Автоматическое сопоставление значений реквизитов информационной базы производится по команде Заполнить. При этом незаполненные значения передаются в сервис, который возвращает найденные соответствующие значения.

Ручное сопоставление значений

Для ручного сопоставления значений необходимо выбрать соответствующее значение из выпадающего списка значений в колонке Значение 1С:Бизнес-сеть.

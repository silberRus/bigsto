Регистрация в сервисе

Это первый шаг для публикации торговых предложений. По команде Зарегистрироваться в сервисе 1С:Бизнес-сеть открывается форма регистрации организаций, где можно зарегистрировать организации в сервисе. Если нет ни одной зарегистрированной организации будет показываться предпупреждение Организации не зарегистрированы.

Подготовка торговых предложений к публикации

По команде Подготовить торговые предложения к публикации открывается форма публикуемых торговых предложений. В этой форме можно добавить новые торговые предложений для публикации, указать дополнительную информацию, сопоставить номенклатуру. Если нет ни одного торгового предложения будет показываться предпупреждение Требуется настройка.

Сопоставление номенклатуры

Для выгрузки торговых предложений обязательно необходимо сопоставить номенклатуру с рубрикатором сервиса 1С:Бизнес-сеть. Если нет номенклатура не сопоставлено будет показываться предпупреждение Номенклатура не сопоставлена.

Проверка публикуемых торговых предложение

По команде открывается отчет о всем публикуемым торговым предложениям с информацией по товарам, ценам, дополнительным реквизитам. Если номенклатура для публикации не найдена, будет показываться предпупреждение Отсутствуют товары для публикации.

Настройка публикации по расписанию

Для автоматической публикации торговых предложений в сервисе необходимо включить флаг Автоматически публиковать и настроить расписание операции.

Выполнение публикации

Команда Опубликовать торговые предложения сейчас выполняет выгрузку торговых предложений в сервис. В случае успешного выполнения будет показываться информация Синхронизация выполнена и время выполнения, в случае возникновения ошибок показывается информация об ошибке. Команда доступна, если во всех предыдущих шагах нет ошибок.

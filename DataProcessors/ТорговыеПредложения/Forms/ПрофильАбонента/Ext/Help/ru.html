Профиль абонента

В данной форме производится настройка регионов профиля абонента для работы с сервисом 1С:Торговые предложения.

Заполнение регионов и адресов

Регионы продаж (для доставки)

Список территорий, в которых поставщик осуществляет реализацию товаров с доставкой товаров покупателям в пределах региона.
Регионы продаж определяют область поиска торговых предложений по региону, указанному покупателем в форме поиска.
Заполняется поставщиком товаров, публикующим торговые предложения.

Адреса складов (для самовывоза)

Адреса складов поставщика, с которых возможен самовывоз товаров покупателями.
Регионы адресов складов определяют область поиска торговых предложений по региону, указанному покупателем в форме поиска.
Заполняется поставщиком товаров, публикующим торговые предложения.

Регионы закупок (для самововоза)

Список территорий где предприятие-покупатель осуществляет закупку товаров.
Регионы закупок определяют область поиска торговых предложений покупателями в режиме поиска "по регионам профиля" в форме поиска.
Заполняется покупателем товаров.

Адреса складов (для доставки)

Адреса складов покупателя на которые можно произвести доставку товаров поставщиками.
Регионы адресов складов определяют область поиска торговых предложений покупателями в режиме поиска "по регионам профиля" в форме поиска.
Заполняется покупателем товаров.

Заполнение регионов и адресов

По команде Заполнить для списков регионов и адресов открывается форма выбора, данные в которых заполняются по контактным данным организаций, складов и т.д. в информационной базе.
При этом, для регионов из контактных данных определяется регион и населенный пункт.

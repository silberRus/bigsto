Назначение обработки

Обработка предназначена для формирования трех текстовых макетов справочника НастройкиХозяйственныхОпераций по текущим данным элементов справочника.

Сам справочник представляет из себя расширение перечисления ХозяйственныеОперации и содержит в себе дополнительную информацию по хозяйственным операциям необходимую:

  • разрешения отображения в иерархической форме выбора перечисления ХозяйственныеОперации в любых реквизитах где вызывается стандартная форма выбора хозяйственной операции;
  • используется в Реестре документов.


Соотношение элементов справочника к значениям перечисления ХозяйственныеОперации один к одному - каждому значению перечисления должен быть установлен в соответствие один элемент справочника.

Макеты НастройкиПредопределенныхЭлементов, СвязанныеДокументы и ФункциональныеОпции содержат в себе значения реквизитов и состав табличных частей Документы и ФункциональныеОпции предопределенных элементов справочника. При добавлении нового предопределенного элемента справочника необходимо вызвать обработчик обновления, который перезаполнит значения реквизитов всех предопределенных элементов (в том числе и новых) - обновит реквизиты по макету. По данным макетов СвязанныеДокументы и ФункциональныеОпции обновится содержание табличных частей предопределенных элементов.

Описание принципа работы

Перед тем как редактировать элементы справочника состав предопределенных элементов справочника должен быть приведен в соответствие с составом значений перечисления ХозяйственныеОперации. При этом имя и синоним предопределенного элемента справочника должны совпадать с именем и синонимом соответствующего значения перечисления.

1. Настроить элементы справочника как это необходимо.
2. Кнопкой Сформировать макеты сохраняет текущие значения реквизитов элементов справочника в макеты.
3. Копированием через буфер поместить данные макетов в метаданные справочника.

Кнопка Заполнить элементы данными из макетов заполняет элементы справочника текущими значениями макетов конфигурации.

Бизнес-сеть

Представляет собой сервисы для взаимодейства пользователей программ 1С:Предприятия. Одним из таких сервисов является упрощенный механизм обмена электронными документами между организациями без использования электронной подписи.

Основные возможности

Основные возможности
  • отправка электронного документа контрагенту без использования Электронной подписи;
  • получение списка входящих документов;
  • загрузка входящего документа в программу;
  • просмотр профиля контрагента.

При обмене электронными документами используются актуальные форматы обмена электронными документами ФНС, CommerceML.

Для подключения программы к сервису 1С:Бизнес-сеть требуется подключить хотя бы одну организацию к сервису.

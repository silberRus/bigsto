Загрузка отчета банка по эквайрингу

Рабочее место предназначено для загрузки в информационную базу данных по операциям эквайринга, направляемых банком-эквайером в виде отчета по предприятию.

Сценарий работы:

  • Выбрать XML файл с данными, содержащий отчет банка по операциям эквайринга.
  • Выбрать статью расходов и подразделение, на которые будет списана сумма эквайринговой комиссии, и при необходимости аналитику расходов.
  • Установить или снять флаг Проводить документы при загрузке.
  • Загрузить отчет банка по эквайрингу.

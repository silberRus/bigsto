Перенести дату поступления

Позволяет перенести дату поступления по правилу для каждой строки.

Варианты переноса:

  • На число года/полугодия/квартала/месяца/декады/недели - позволяет перенести на указанный номер дня от начала периода. Если указано 31 число месяца - будет установлен последний день месяца.
  • На первый день недели, например Вторник, позволяет перенести на первый Вторник в периоде плана каждую строку. Дата в каждом периоде зависит от выбранного дня недели.
  • На последний день недели, аналогично переносу на первый день.

Распределить количество

Позволяет распределить количество в рамках текущего периода планирования.

Варианты распределения периода:

  • По полугодиям - количество будет распределено на 2 части. Даты поступления будут установлены на начало каждого полугодия.
  • По кварталам - количество будет распределно на 2 части, если периодичность сценария "Полугодие" или на 4 части, если периодичность сценария год.
  • По месяцам - распределение на 12, 6 или 3 части в зависимости от периодичности сценария.
  • По декадам - даты поступления будут установлены на 1, 11 и 21 числа в одном периоде планирования.
  • По неделям - количество недель зависит от периодичности сценария и выбранного дня недели для поступления.
  • По количество периодов - указывается произвольное количество частей распределение. Но не может быть больше количества дней в периода планирования.

Способ распределения количества в периоде:

  • Пропорционально - количество будет разделено пропорционально в периоде.
  • По долям - на следующей странице необходимо указать доли для каждого периода распределения.

Если доля установлена = 0, то этот период при распределении пропускается. Аналогично, если результат округления после распределения по долям равен 0.

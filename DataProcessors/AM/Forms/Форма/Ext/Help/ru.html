Поиск артикула

В данной области окна задаются условия для поиска информации по артикулу.
Артикул заполнен, если был выбран в списке Детали узла для выделенной детали пункт Открыть в Laximo.AM.

Артикул можно и ввести в поле ввода, но он должен присутствовать обязательно!

Набор следующих чек-боксов позволяет задать условия для отображения результата поиска:

  • Получить кроссы, замены детали - при установленном чек-боксе найденные кроссы, замены будут отображаться на закладке Замены/Аналоги;
  • Дополнительные наименования - при установленном чек-боксе дополнительные наименования детали для указанного артикула отображаются на закладке Свойства;
  • Получить изображения деталей - при установленном чек-боксе информация об изображении детали с указанным артикулам отображается на закладке Изображение;
  • Получить веса деталей - при установленном чек-боксе дополнительные веса детали для указанного артикула отображаются на закладке Свойства;
  • Нужны свойства деталей - при установленном чек-боксе свойства детали для указанного артикула отображаются на закладке Свойства.

По кнопке Получить информацию отображается результат поиска.

Результат поиска

Список включает найденные по артикулы детали,содержит следующую информацию:

  • N - порядковый номер в списке;
  • № по каталогу - OEM;
  • Производитель - бренд производителя;
  • Наименование - наименование детали;
  • Вес в кг - вес детали в килограммах.


Для выделенной строки списка отображается Информация о детали.

Информация о детали

Закладка "Замены/Аналоги"

Закладка доступна, если установлен чек-бокс Получить кроссы, замены детали.
Список включает замены/дубликаты, содержит следующую информацию:

  • N - порядковый номер в списке;
  • Тип замены - тип связи, см. описание типов связи
  • Направление - направление связи;
  • № по каталогу - OEM;
  • Производитель - бренд производителя;
  • Наименование - наименование детали;
  • Рейтинг - рейтинг достоверности по пятибалльной шкале.

Закладка "Свойства"

Закладка доступна, если установлен чек-бокс Получить свойства деталей.
Список включает набор свойств детали, содержит следующую информацию:

  • N - порядковый номер в списке;
  • Свойство - наименование свойства;
  • Значение свойства - значение свойства;
  • Язык - код языка на котором отображается значение свойства;
  • Рейтинг -рейтинг достоверности.

Закладка "Изображение"

Закладка доступна, если установлен чек-бокс Получить изображения деталей.
Список рисунков с указанием:

  • N - порядковый номер в списке;
  • URL Картинки - адрес на страницу с изображением рисунка.

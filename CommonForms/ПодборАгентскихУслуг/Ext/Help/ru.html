Подбор агентских услуг

Форма используется для подбора агентских услуг в табличную часть Соглашения об условиях закупок. При открытии формы по умолчанию устанавливается отбор по партнеру (принципалу) из соглашения. В табличную часть выводятся только агентские услуги данного принципала. Выбор услуг осуществляется путем установления флажков у соответствующих строк. После нажатия на кнопку Перенести в справочник данная форма будет закрыта, а результат выбора будет помещен в соглашение с поставщиком
